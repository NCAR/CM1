Sample submission scripts for NCAR's cheyenne (SGI ICE XA Cluster):

  • cm1run_cheyenne: multiple processors using MPI (distributed memory) (cheyenne)
  • cm1run_sp_cheyenne: single processor

See Running Jobs for more information about using cheyenne.

Sample submission scripts for NCAR's yellowstone (IBM iDataPlex with Intel Sandy Bridge processors):

  • cm1run_sp: single processor (caldera)
  • cm1comp: script for compiling CM1 on caldera (just place cm1comp in the "src" directory and submit from there) (caldera)
  • cm1run_mpi: multiple processors using MPI (distributed memory) (yellowstone)

See Yellowstone User Guide for more information; in particular, see the section on Running Jobs.
