Sample submission scripts for NCAR's bluefire (IBM POWER6):

  • cm1run_sp: single processor
  • cm1run_omp: OpenMP
  • cm1run_mpi: MPI

See Bluefire Quick Start Guide for more information.

Note for users that are switching from bluevista to bluefire: You can use the same compiler flags that you used for bluevista. (No changes in the Makefile should be necessary.) However, you definitely need to make changes to your submission scripts for maximum performance on bluefire; the scripts above are recommended.
